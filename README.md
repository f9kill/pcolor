|-------------------|-------------------|

| BACK_RED          | FONT_RED          |
| BACK_PURPLE       | FONT_PURPLE       |
| BACK_YELLOW       | FONT_YELLOW       |
| BACK_WHITE        | FONT_WHITE        |
| BACK_GRAY         | FONT_GRAY         |
| BACK_LIGHT_BLUE   | FONT_LIGHT_BLUE   |
| BACK_LIGHT_GREEN  | FONT_LIGHT_GREEN  |
| BACK_LIGHT_AQUA   | FONT_LIGHT_AQUA   |
